## ⚡ Veja Seu Aplicativo (Prévia Visual)

        
Vamos dar uma olhada em como seu aplicativo pode parecer! O V0.dev é uma ferramenta gratuita da Vercel que transforma descrições de texto em designs de interface de usuário reais.


        
            
Seu prompt pronto para o V0.dev:

            📋 CopiarCrie a interface de usuário para um aplicativo de geração de conteúdo científico.
O aplicativo se chama "AcademIA" (ou algo similar, sinta-se à vontade para sugerir).
Ele deve permitir ao usuário informar um tema e gerar Trabalhos de Conclusão de Curso (TCC), Estudos de Caso e Artigos científicos.
A interface deve ser moderna, limpa e profissional, com um esquema de cores que remeta a tons de azul e cinza, com toques de verde para ações de sucesso.
Use o framework React para o frontend e o framework de estilos Tailwind CSS.
Inclua:
1. Uma página inicial com um campo de entrada grande para o "Tema" e um botão "Gerar Conteúdo".
2. Opções para selecionar o tipo de conteúdo a ser gerado (TCC, Estudo de Caso, Artigo Científico) usando botões de rádio ou um dropdown.
3. Uma área para exibir o progresso da geração.
4. Uma área para exibir o conteúdo gerado, com opções para copiar ou baixar.
5. Um cabeçalho simples com o nome do aplicativo e talvez um ícone.
O design deve ser responsivo para desktop e mobile.
            
**Siga estes passos:**

            
                - Vá para v0.dev.
                - Clique no botão que diz "Start Chatting" (Iniciar Conversa).
                - Cole o prompt que você copiou acima na caixa de texto.
                - Aguarde e veja a interface do seu aplicativo aparecer!
            
        
        ✅Checkpoint: Você deve ver uma prévia ao vivo da interface do seu aplicativo.
    

    
        
## 🖥️ Configure Sua Ferramenta de Codificação com IA

        
Para transformar esses planos em um aplicativo real, você vai precisar de um editor de código. Uma ferramenta de codificação com IA pode ser como ter um assistente superinteligente ao seu lado. Embora você não tenha escolhido um editor de IA específico, eu recomendo o Cursor como uma excelente opção para começar, pois ele é feito para trabalhar com IA.


        
            
                1
                
### Escolha e Baixe Seu Editor de Código

            
            
Você pode usar qualquer editor de código que preferir (como o VS Code, por exemplo), mas se quiser aproveitar o poder da IA, o Cursor é uma ótima escolha. O Windsurf é outra alternativa interessante.

            

                - **Para Cursor:** Vá para cursor.sh e clique no botão de download para o seu sistema operacional. Siga as instruções para instalar.
                - **Para Windsurf:** Vá para windsurf.ai e siga as instruções de instalação.
                - **Para outros editores (como VS Code) e IA de chat:** Se você preferir usar um editor de código tradicional e um chat de IA separado (como ChatGPT, Claude, etc.), você pode usar as "Instruções Universais" que você encontrará na aba de "Implementação" do seu PRD Stacks.
            

        

        
            
                2
                
### Crie a Pasta do Seu Projeto

            
            
É como criar um novo lugar na sua mesa de trabalho para o seu projeto. Crie uma pasta vazia em algum lugar fácil de encontrar no seu computador, por exemplo, "AcademIA".

            
**Dica:** Escolha um nome simples e sem espaços para a pasta.

        

        
            
                3
                
### Abra a Pasta no Seu Editor de Código

            
            
Depois de instalar o Cursor (ou Windsurf/VS Code):

            

                - Abra o Cursor (ou o editor que você escolheu).
                - Vá em "File" (Arquivo) no menu superior.
                - Clique em "Open Folder..." (Abrir Pasta...) ou "Open Project..." (Abrir Projeto...).
                - Navegue até a pasta "AcademIA" que você criou e selecione-a.
            

        

        
            
                4
                
### Crie o Arquivo de Regras para a IA (se usar Cursor/Windsurf)

            
            
Este arquivo é como um "manual de instruções" para o seu assistente de IA, dizendo a ele como você gosta de trabalhar e quais são as regras do projeto.

            

                - No seu editor de código (Cursor ou Windsurf), no painel lateral esquerdo (onde você vê os arquivos), clique com o botão direito na área vazia ou no nome da sua pasta "AcademIA".
                - Selecione "New File" (Novo Arquivo).
                Digite o nome do arquivo:
                    📋 Copiar`.cursorrules`
                    (se estiver usando Cursor) ou
                    📋 Copiar`.windsurfrules`
                    (se estiver usando Windsurf).
                
                - Pressione Enter. Agora você tem um arquivo vazio.
                - Vá para a aba "Implementation" (Implementação) no seu PRD Stacks, encontre a seção "IDE Rules" (Regras do IDE) e copie todo o conteúdo.
                - Cole esse conteúdo no arquivo `.cursorrules` (ou `.windsurfrules`) que você acabou de criar e salve o arquivo (Ctrl+S ou Cmd+S).
            

        
        ✅Checkpoint: Seu editor de código está aberto com a pasta do seu projeto e o arquivo de regras da IA (.cursorrules ou .windsurfrules) está criado e preenchido.
    

    
        
## 🗄️ Configure Seu Banco de Dados

        
Seu aplicativo precisa de um lugar para guardar todas as informações, como os temas pesquisados, os artigos encontrados e o conteúdo gerado. Esse lugar é o banco de dados. Você escolheu o Supabase, que é como um "Google Docs" para seus dados, mas muito mais poderoso e seguro!


        
            
                1
                
### Crie Sua Conta e Projeto no Supabase

            
            
                - Vá para supabase.com.
                - Clique em "Start your project" (Iniciar seu projeto) ou "Sign In" (Entrar) no canto superior direito.
                - Crie uma conta usando seu e-mail ou GitHub.
                - Após fazer login, você será direcionado para o seu painel. Clique em "New Project" (Novo Projeto).
                - Escolha um nome para o seu projeto (ex: "AcademIA-DB"), defina uma senha forte para o banco de dados e selecione a região mais próxima de você.
                - Clique em "Create new project" (Criar novo projeto). Isso pode levar alguns minutos.
            
        

        
            
                2
                
### Execute o Script SQL para Criar Suas Tabelas

            
            
Agora vamos dizer ao Supabase como organizar seus dados. Pense nisso como dar as instruções para montar as prateleiras e gavetas no seu armário de dados.

            
                - No painel do Supabase, no menu lateral esquerdo, procure um ícone que parece um pergaminho ou uma folha de papel com um "" (símbolo de código). Clique nele. É o "SQL Editor" (Editor SQL).
                - Você verá uma área onde pode digitar comandos de SQL.
                - Vá para a aba "SQL Guide" (Guia SQL) no seu PRD Stacks e copie todo o script SQL que você encontrar lá.
                - Cole esse script no "SQL Editor" do Supabase.
                - Clique no botão "Run" (Executar), geralmente no canto superior direito do editor.
            
        
        ✅Checkpoint: Você deve ver uma mensagem de sucesso no Supabase e, se for para a seção "Table Editor" (Editor de Tabelas) no menu lateral, verá suas tabelas listadas (ex: "temas", "artigos", "conteudo_gerado").
        🆘Stuck? Se você vir um erro vermelho, certifique-se de que copiou TODO o texto do script SQL, incluindo a primeira linha. Às vezes, um pequeno pedaço faltando pode causar problemas. Tente copiar e colar novamente.
    

    
        
## 🚀 Dê o Primeiro Comando à Sua IA

        
Este é o momento mágico! Agora você vai dar a primeira instrução ao seu assistente de IA (ou à ferramenta de chat de IA que você escolheu) para começar a construir o seu aplicativo. Pense nisso como dar as chaves para um construtor experiente e dizer "comece por aqui!".


        
            
                1
                
### O Primeiro Comando

            
            
No seu editor de código (se estiver usando Cursor ou Windsurf, use a janela de chat da IA integrada) ou na sua ferramenta de chat de IA (como ChatGPT, Claude), copie e cole o seguinte prompt:

            📋 CopiarOlá, sou um desenvolvedor júnior e preciso da sua ajuda para construir meu aplicativo "AcademIA".
Meu objetivo é criar um programa para gerar TCC, Estudo de Caso e Artigos científicos a partir de um tema informado, buscando os 10 melhores e mais recentes artigos/documentos relevantes, montando uma memória e gerando conteúdo autoral em formato científico.

Minha stack de tecnologia é:
- Frontend: React (com Vite para bundling, Tailwind CSS para estilização, React Router v6+ para roteamento)
- Backend: Python (com FastAPI para API assíncrona, Pydantic para validação)
- Banco de Dados: Supabase (PostgreSQL, com Supabase Auth, RLS Policies, Edge Functions)
- Infraestrutura: Vercel (para deploy do frontend e funções serverless)

Por favor, comece configurando o projeto.
1. Crie a estrutura básica do projeto React com Vite e configure o Tailwind CSS.
2. Crie a estrutura básica do projeto Python com FastAPI.
3. Adicione um arquivo .env para variáveis de ambiente.
4. Crie um arquivo README.md explicando como rodar o frontend e o backend.

Lembre-se de seguir as melhores práticas para cada tecnologia e de usar as instruções que estão no arquivo de regras (se aplicável).
            
**O que acontece agora?**

            
Se você estiver usando o Cursor ou Windsurf, a IA lerá automaticamente o arquivo `.cursorrules` (ou `.windsurfrules`) que você criou e começará a escrever os arquivos de código diretamente na sua pasta do projeto. Se estiver usando uma IA de chat separada, ela te dará os códigos para você copiar e colar.

        
        ✅Checkpoint: Você deve ver a IA escrevendo arquivos de código na sua pasta do projeto (ou fornecendo blocos de código para você copiar).
        🆘Stuck? Se a IA parecer confusa ou não começar a escrever código, tente dizer: "Leia o arquivo de regras primeiro, depois comece com o passo 1." ou "Por favor, me dê o código para o passo 1.".
    

    
        
## 📱 O Que Vem Agora?

        
Você deu os primeiros passos e tem uma base sólida para o seu aplicativo! Agora, vamos ver o que você pode fazer a seguir para continuar construindo e lançando seu projeto.


        

            
                **Veja Seu Aplicativo Rodando:** Depois que a IA configurar o projeto, você poderá ver seu aplicativo funcionando no seu computador. Abra o "terminal" (é aquela janela de texto preta onde os programadores digitam comandos) na pasta do seu projeto e digite `npm run dev` para o frontend e o comando que a IA te der para o backend. O terminal vai te dizer qual endereço (geralmente http://localhost:3000) abrir no seu navegador para ver seu app.
            
            
                **Coloque Seu App Online:** Quando estiver pronto para mostrar seu aplicativo ao mundo, você pode "publicá-lo" usando serviços como o Vercel (para o frontend React) e o próprio Supabase (para o backend Python com Edge Functions). A IA pode te ajudar com os comandos de deploy!
            
            
                **Conquiste Seus Primeiros Usuários:** Não se esqueça de que construir é só o começo! Pense em como você vai divulgar seu aplicativo e atrair pessoas para usá-lo. As ideias do seu "Market Analysis" (se você gerou uma) podem ser um ótimo ponto de partida.
            
            
                **Continue Melhorando:** A beleza de usar IA é que você pode iterar rapidamente. Volte para o PRD Stacks, atualize seu PRD com novas ideias ou mudanças, e gere novas seções para continuar evoluindo seu aplicativo.
